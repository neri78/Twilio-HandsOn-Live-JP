## 必要・推奨環境
このハンズオンを実施する場合は、次の環境が必要となります。

- Twilioアカウント [アカウントの作成方法](https://www.twilio.com/blog/how-to-create-twilio-account-jp)
- [Node.js (v14以降）](https://nodejs.org/ja/)
- 任意のコードエディタ
